## Conventions

- **Cross-link, never copy.** If information lives in a story or CLAUDE.md, link to it rather than duplicating.
- **Decisions embedded.** Architecture decisions are inline in the relevant section, not in separate ADR files.
- Each article has `Related stories:` and `Related files:` links in the header.
- Articles are updated when the implementation changes. The `_Last updated:` footer tracks freshness.
