# Эпик 07: Полный опросник онбординга

## Описание

Расширить поток онбординга за пределы одного вопроса об уровне словарного запаса до многошагового опросника, охватывающего возраст ребёнка, текущие навыки по вербальным оперантам, способы коммуникации (речь, PECS, жесты) и цели родителей. Использовать ответы для персонализации начального пути оценки и конфигурации словаря.
