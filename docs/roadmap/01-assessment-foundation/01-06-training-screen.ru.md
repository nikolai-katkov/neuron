# 01-06: Экран тренировки

Статус: Готово

## Описание

Экран с руководством, показываемый после оценки "Нет". Обучает родителя тому, как развивать конкретный навык, используя прогрессивное раскрытие: заглушка видео, краткое пошаговое руководство видно по умолчанию, полное руководство свёрнуто. "Повторить" возвращает к экрану оценки для повторной проверки. "Назад" возвращает к списку критериев.

## Критерии приёмки для пользователя

- [x] Место для видео отображает статичное изображение-заглушку с надписью "Видео скоро будет"
- [x] Краткое руководство видно по умолчанию: пронумерованные шаги (например, 1. Покажите 2 предмета, 2. Подождите, 3. Подскажите при необходимости, 4. Подкрепите)
- [x] "Подробнее" разворачивает полное руководство (детальное объяснение, распространённые ошибки, дополнительные примеры)
- [x] Сворачивание "Подробнее" скрывает полное руководство
- [x] Кнопка "Повторить" переходит назад к экрану оценки для этого критерия
- [x] Кнопка "Назад" возвращает к списку критериев
- [x] Тренировочный контент специфичен для критерия (не общий)

## Системные критерии приёмки

- [x] Маршрут: `/sections/:sectionId/criteria/:criterionId/train`
- [x] Тренировочный контент загружается из отдельной модели данных тренировки (не из данных критерия)
- [x] "Повторить" переходит к экрану оценки с сохранением контекста критерия
- [x] "Назад" переходит к списку критериев
- [x] Критерий остаётся в статусе InProgress во время цикла тренировки
- [x] Нет тупиков: обе кнопки всегда видны и функциональны

## Критерии приёмки для разработки

- [x] Прогрессивное раскрытие использует CSS-переходы (согласовано с паттерном экрана введения)
- [x] CSS Modules только с дизайн-токенами
- [x] Адаптивность для всех профилей устройств
- [x] Повторное использование компонента/паттерна прогрессивного раскрытия из 01-03, если применимо (правило двух -- следить за третьим использованием)
- [x] Юнит-тест: поведение переключателя "Подробнее"
- [x] Юнит-тест: корректные данные тренировки загружаются на основе параметров маршрута
- [x] E2E-тест: переход из оценки "Нет", проверка тренировочного контента, нажатие "Повторить" возвращает к оценке
- [x] E2E-тест: "Назад" возвращает к списку критериев
