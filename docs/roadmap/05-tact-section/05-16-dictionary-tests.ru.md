# 05-16: Тесты словаря

Статус: Готово

## Описание

Добавить юнит-тесты для хелперов модели данных словаря, валидации данных словаря и хука `useDictionary` (онбординг, включение слов, отслеживание освоения, прогресс категорий, слова для тренировки, сохранение в localStorage).

## Системные критерии приёмки

- [x] `tests/data/dictionary.test.ts`:
  - `getWordId` формирует формат `category:difficulty:index`
  - `parseWordId` разбирает валидные ID и возвращает null для невалидных
  - Обратное преобразование: `parseWordId(getWordId(...))` возвращает исходные значения
  - `getCriterionCategoryIds` возвращает мотивирующие для tact/mand 1--2, бытовые для tact-3, null для высших уровней, null для неподдерживаемых разделов
  - `getVocabularyWord` разрешает валидные ID в текст слова, возвращает null для невалидных/отсутствующих
- [x] `tests/data/vocabulary.test.ts`:
  - Оба языка имеют ровно 27 категорий
  - Все категории привязаны к разделу `tact`
  - Уникальные ID категорий по языку
  - Непустые названия для каждой категории
  - Непустые массивы слов на всех трёх уровнях сложности
  - Одинаковые ID категорий в обоих языках
  - Одинаковое количество слов по сложности и категориям в обоих языках
- [x] `tests/hooks/useDictionary.test.tsx`:
  - Выбрасывает ошибку при использовании вне `DictionaryProvider`
  - Начинается с незавершённым онбордингом
  - `completeOnboarding` устанавливает уровень и отмечает завершение
  - Beginner: включает только простые слова из мотивирующих категорий
  - Intermediate: включает все простые слова
  - Advanced: включает простые + средние слова
  - `setWordInclusion` переключает статус включения слова
  - `setWordMastery` переключает освоение по операнту
  - `getCategoryProgress` возвращает корректные счётчики включённых/всего
  - `getWordState` возвращает excluded + none для невалидных ID слов
  - `getCategoryProgress` возвращает нули для неизвестной категории
  - `getPracticeWords` возвращает включённые слова, отфильтрованные по критерию
  - `getPracticeWords` возвращает больше слов для высших уровней, чем для низших
  - Состояние сохраняется в localStorage между перерисовками
