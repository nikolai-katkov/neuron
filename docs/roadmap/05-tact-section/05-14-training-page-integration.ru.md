# 05-14: Интеграция страницы тренировки (слова для практики)

Статус: Готово

## Описание

Интегрировать персональный словарь в страницу тренировки. После пошагового руководства отображать раздел "Слова для тренировки" с включёнными словами ребёнка, релевантными текущему критерию, с заглушками изображений.

## Критерии приёмки для пользователя

- [x] Страница тренировки показывает раздел "Слова для тренировки" под пошаговым руководством
- [x] Заголовок раздела: "Слова для тренировки" с подсказкой "Из словаря вашего ребёнка"
- [x] Слова отображаются как карточки в адаптивной сетке, каждая с заглушкой изображения (иконка Image) и текстом слова
- [x] Отображаются только слова, отмеченные как "включённые" в словаре
- [x] Слова фильтруются по релевантности критерию:
  - Такт/Манд уровни 1--2: только мотивирующие категории (игрушки, домашние животные, еда, фрукты, части тела, транспорт)
  - Такт уровень 3: только бытовые категории (одежда, обувь, мебель, посуда, электроприборы, головные уборы, школьные принадлежности)
  - Такт уровни 4--5 и Манд уровни 3--5: все включённые слова
- [x] Раздел скрыт, если нет включённых слов, подходящих критерию

## Системные критерии приёмки

- [x] Компонент `PracticeWordsSection` в `TrainingPage.tsx` принимает проп `criterionId`
- [x] Вызывает `useDictionary().getPracticeWords(criterionId)` для получения отфильтрованных включённых слов
- [x] Субкомпонент `PracticeWordCard` рендерит каждое слово с заглушкой изображения и подписью
- [x] CSS-классы: `practiceSection`, `practiceHeader`, `practiceTitle`, `practiceHint`, `practiceGrid`, `wordCard`, `wordImage`, `wordLabel`
- [x] Ключи переводов: `practiceWords`, `practiceWordsHint`
