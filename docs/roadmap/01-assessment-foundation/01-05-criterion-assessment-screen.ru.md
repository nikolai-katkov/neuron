# 01-05: Экран оценки критерия

Статус: Готово

## Описание

Экран бинарной оценки для одного критерия. Представляет чёткий вопрос Да/Нет с иллюстрацией и контекстной информацией. "Да" отмечает критерий как завершённый и возвращает к списку критериев. "Нет" переходит к экрану тренировки.

## Критерии приёмки для пользователя

- [x] Чёткий бинарный вопрос отображается на видном месте (например, "Может ли ребёнок попросить один из двух предметов?")
- [x] Область иллюстрации показывает статичное изображение-заглушку, релевантное критерию (2 различных объекта для критериев МАНД)
- [x] Раздел контекста показывает: краткое объяснение того, на что обращать внимание, условия из критерия
- [x] Две кнопки: "Да" и "Нет" -- без двусмысленности, без других вариантов
- [x] "Да" показывает краткую положительную обратную связь, отмечает выполнение, возвращает к списку критериев с обновлённым статусом
- [x] "Нет" переходит к экрану тренировки для этого критерия
- [x] Условия критерия (например, "подсказки не допускаются") видны, чтобы родитель оценивал правильно

## Системные критерии приёмки

- [x] Маршрут: `/sections/:sectionId/criteria/:criterionId/assess`
- [x] "Да" сохраняет балл (1) и обновляет статус критерия на Completed
- [x] "Нет" сохраняет балл (0) и обновляет статус критерия на InProgress
- [x] Балл хранится как числовое значение (0 или 1), не булево (см. [Вопросы PM: Модель оценивания](../../knowledge/pm-questions-scoring-model.ru.md))
- [x] Состояние сохраняется между перезагрузками страницы
- [x] Навигация: Да -> список критериев, Нет -> экран тренировки

## Критерии приёмки для разработки

- [x] CSS Modules только с дизайн-токенами
- [x] Адаптивность для всех профилей устройств
- [x] Изображения-заглушки используют единое соотношение сторон и стиль по умолчанию
- [x] Юнит-тест: обработчики Да/Нет корректно обновляют состояние
- [x] Юнит-тест: корректные данные критерия загружаются на основе параметров маршрута
- [x] E2E-тест: полный поток "Да" (нажатие Да -> возврат к списку с обновлённым состоянием)
- [x] E2E-тест: полный поток "Нет" (нажатие Нет -> переход к экрану тренировки)
