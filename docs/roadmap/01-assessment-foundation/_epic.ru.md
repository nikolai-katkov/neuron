# Эпик 01: Платформа оценки

## Описание

Реализация платформы оценки: список разделов, введение, прогрессия критериев, бинарная оценка, руководство по тренировке и цикл оценка-тренировка-переоценка. МАНД используется как первый раздел; ТАКТ отображается как заглушка.

**Поток:** Изучение > Оценка > (при необходимости) Тренировка > Переоценка
