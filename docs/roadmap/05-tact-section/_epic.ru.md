# Эпик 05: Раздел Такт

## Описание

Активация раздела Такт (Называние) с полным тренировочным контентом, системой словаря (27 категорий x 3 уровня сложности), персональным словарём с онбордингом и 7 заглушками разделов для будущей работы. Расширение модели данных типами словаря и словарного запаса.

**Поток:** Изучение > Оценка > (при необходимости) Тренировка > Переоценка
