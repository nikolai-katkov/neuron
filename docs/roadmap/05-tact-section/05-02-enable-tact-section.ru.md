# 05-02: Активация раздела ТАКТ

Статус: В ожидании

## Описание

Установить `isAvailable` в `true` для раздела ТАКТ, исправить русский подзаголовок, убрать примечание "скоро появится" из текста введения.

## Критерии приёмки для пользователя

- [ ] Карточка ТАКТ в списке разделов кликабельна и ведёт на страницу введения
- [ ] Русский подзаголовок показывает "Называние" (вместо "Обозначение")
- [ ] Страница введения ТАКТ показывает полное объяснение без примечания "скоро появится"

## Системные критерии приёмки

- [ ] `sections.ts`: ТАКТ `isAvailable` установлен в `true` для обоих языков
- [ ] `sections.ts`: русский подзаголовок изменён на "Называние"
- [ ] `introduction.ts`: абзац "скоро появится" удалён из EN и RU

## Связанные файлы

- `src/i18n/translations/sections.ts`
- `src/i18n/translations/introduction.ts`
