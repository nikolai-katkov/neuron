# 05-06: Обновление документации

Статус: В ожидании

## Описание

Обновить документацию знаний с методологией тренировки ТАКТ, документацией системы словаря и обновлённым объёмом MVP. Использовать профессиональную ABA-терминологию.

## Критерии приёмки для пользователя

- [ ] Методология тренировки ТАКТ задокументирована (существительные, глаголы, метод отвлечения, жетонное подкрепление, cold probe)
- [ ] Система словаря задокументирована (27 категорий, модель сложности)
- [ ] Объём MVP обновлён: ТАКТ в "Включено", а не "Исключено"
- [ ] Обе языковые версии обновлены
