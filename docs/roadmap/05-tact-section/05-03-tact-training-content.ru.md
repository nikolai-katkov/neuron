# 05-03: Тренировочный контент Такт

Статус: Готово

## Описание

Добавить тренировочный контент для всех 5 критериев Такт (tact-1 -- tact-5) на обоих языках. Каждая запись включает 4-шаговое краткое руководство, полное руководство и массив типичных ошибок.

## Критерии приёмки для пользователя

- [x] Страница тренировки каждого критерия Такт показывает 4-шаговое краткое руководство
- [x] "Подробнее" раскрывает полное руководство
- [x] Раздел типичных ошибок показывает 4 пункта на критерий
- [x] Контент написан для родителей: тёплый, практический, без клинического жаргона
- [x] Русский контент использует устоявшуюся ABA-терминологию

## Системные критерии приёмки

- [x] `training.ts` содержит ключи `tact-1` -- `tact-5` на обоих языках
- [x] Каждая запись имеет `criterionId`, совпадающий с ключом
- [x] Каждая запись имеет ровно 4 шага в `shortGuide`
- [x] Каждая запись имеет непустые `fullGuide` и `commonMistakes` (4 пункта)
