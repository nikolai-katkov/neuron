# Эпик 08: Изображения слов и тренировочные материалы

## Описание

Заменить заглушки изображений в карточках слов словаря и на страницах тренировки реальными изображениями. Подготовить или сгенерировать карточки для всех слов словаря по 27 категориям. Добавить управление изображениями, ленивую загрузку и доставку через CDN.
