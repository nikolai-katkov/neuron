# Эпик 06: Интеграция оценки и словаря

## Описание

Связать поток оценки с персональным словарём, чтобы результаты оценки влияли на состояние словаря и наоборот. Автоматически корректировать включение и освоение слов на основе завершения критериев, а также показывать рекомендации на основе словаря во время оценки.
