# Эпик 09: Аналитика словаря

## Описание

Добавить отслеживание прогресса и аналитику для персонального словаря: рост словарного запаса во времени, уровни освоения по вербальным оперантам, метрики завершения категорий и визуальные панели прогресса. Дать родителям возможность видеть траекторию развития словарного запаса ребёнка.
