# 01-02: Экран списка разделов

Статус: Готово

## Описание

Точка входа в приложение, показывающая доступные разделы навыков в виде карточек. МАНД активен с отслеживанием прогресса. ТАКТ отображается как неактивная заглушка "скоро будет". Нажатие на МАНД переходит к экрану введения в МАНД.

## Критерии приёмки для пользователя

- [x] Карточка МАНД отображает: заголовок "МАНД", подзаголовок "Просьбы", статус (Не начато / В процессе / Завершено), прогресс (X/5)
- [x] Карточка ТАКТ отображает: заголовок "ТАКТ", подзаголовок "Скоро будет", визуально неактивна/затемнена
- [x] Нажатие на карточку МАНД переходит к экрану введения в МАНД
- [x] Нажатие на карточку ТАКТ не выполняет действий (неактивна)
- [x] Обновления прогресса отражают фактическое состояние оценки (сохраняется между сеансами)

## Системные критерии приёмки

- [x] Маршрут: `/` (точка входа в приложение)
- [x] Прогресс раздела вычисляется из состояний критериев (количество завершённых из общего числа)
- [x] Статус раздела определяется: все NotStarted = "Не начато", любой InProgress или смешанный = "В процессе", все Completed = "Завершено"
- [x] Состояние сохраняется между перезагрузками страницы (localStorage или аналог)

## Критерии приёмки для разработки

- [x] CSS Modules только с дизайн-токенами (без захардкоженных значений)
- [x] Адаптивность для десктопа (1920x1080), планшета (768x1024), мобильного (375x667)
- [x] Юнит-тест для логики вычисления прогресса/статуса
- [x] E2E-тест: загрузка приложения, проверка видимости карточки МАНД, нажатие переходит к введению
