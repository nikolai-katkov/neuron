# 05-02: Активация раздела Такт

Статус: Готово

## Описание

Установить `isAvailable` в `true` для раздела Такт, исправить русский подзаголовок, убрать примечание "скоро появится" из текста введения.

## Критерии приёмки для пользователя

- [x] Карточка Такт в списке разделов кликабельна и ведёт на страницу введения
- [x] Русский подзаголовок показывает "Называние" (вместо "Обозначение")
- [x] Страница введения Такт показывает полное объяснение без примечания "скоро появится"

## Системные критерии приёмки

- [x] `sections.ts`: Такт `isAvailable` установлен в `true` для обоих языков
- [x] `sections.ts`: русский подзаголовок изменён на "Называние"
- [x] `introduction.ts`: абзац "скоро появится" удалён из EN и RU

## Связанные файлы

- `src/i18n/translations/sections.ts`
- `src/i18n/translations/introduction.ts`
