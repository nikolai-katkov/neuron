# 05-12: Страница деталей категории словаря

Статус: Готово

## Описание

Добавить маршрут `/dictionary/:categoryId`, показывающий все слова категории, организованные по уровням сложности. Слова можно переключать между включёнными/исключёнными, у каждого слова есть точки освоения по оперантам. На десктопе -- три колонки рядом; на мобильном -- вкладки для переключения между уровнями сложности. Перетаскивание перемещает слова между зонами включения.

## Критерии приёмки для пользователя

- [x] Заголовок страницы -- название категории на текущем языке
- [x] Хлебные крошки показывают: Главная > Словарь ребёнка > [Название категории]
- [x] Каждый уровень сложности (Простые, Средние, Сложные) показывает две зоны: "В словаре" и "Не включены"
- [x] Слова в зоне "В словаре" показывают четыре точки освоения (Манд, Такт, Понимание, Эхоик) -- нажатие на точку переключает освоение
- [x] Нажатие на строку слова переключает его между включённым и исключённым статусами
- [x] Слова перетаскиваются между двумя зонами через drag-and-drop
- [x] На десктопе (широкий экран): все три колонки сложности отображаются рядом
- [x] На мобильном (узкий экран): панель вкладок переключает между уровнями сложности, показывая одну колонку за раз
- [x] Переход на несуществующий categoryId перенаправляет на `/dictionary`

## Системные критерии приёмки

- [x] Компонент `DictionaryCategoryDetailPage` в `src/pages/DictionaryCategoryDetailPage.tsx`
- [x] Маршрут зарегистрирован в `App.tsx` по пути `/dictionary/:categoryId`
- [x] Использует компонент `Tabs` из `src/components/ui/` для мобильного переключения сложности
- [x] Использует `useDictionary()` для `getWordState`, `setWordInclusion`, `setWordMastery`
- [x] Субкомпонент `MasteryDot` рендерит кнопки-переключатели по оперантам с `aria-label`
- [x] Субкомпонент `WordRow` обрабатывает переключение по клику, начало перетаскивания и точки освоения
- [x] Субкомпонент `DifficultyColumn` рендерит зоны включённых/исключённых с обработчиками drag-over/drop
- [x] CSS-модуль с классами `desktopColumns` (рядом) и `mobileTabs` (скрыт на десктопе)
