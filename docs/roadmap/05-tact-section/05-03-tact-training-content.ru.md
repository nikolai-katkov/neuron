# 05-03: Тренировочный контент ТАКТ

Статус: В ожидании

## Описание

Добавить тренировочный контент для всех 5 критериев ТАКТ (tact-1 -- tact-5) на обоих языках. Каждая запись включает 4-шаговое краткое руководство, полное руководство и массив типичных ошибок.

## Критерии приёмки для пользователя

- [ ] Страница тренировки каждого критерия ТАКТ показывает 4-шаговое краткое руководство
- [ ] "Подробнее" раскрывает полное руководство
- [ ] Раздел типичных ошибок показывает 4 пункта на критерий
- [ ] Контент написан для родителей: тёплый, практический, без клинического жаргона
- [ ] Русский контент использует устоявшуюся ABA-терминологию

## Системные критерии приёмки

- [ ] `training.ts` содержит ключи `tact-1` -- `tact-5` на обоих языках
- [ ] Каждая запись имеет `criterionId`, совпадающий с ключом
- [ ] Каждая запись имеет ровно 4 шага в `shortGuide`
- [ ] Каждая запись имеет непустые `fullGuide` и `commonMistakes` (4 пункта)
