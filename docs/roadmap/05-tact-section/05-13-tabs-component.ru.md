# 05-13: UI-компонент Tabs

Статус: Готово

## Описание

Добавить универсальный компонент `Tabs` в дизайн-систему для переключения между представлениями. Используется страницей деталей категории словаря для мобильного переключения сложности.

## Критерии приёмки для пользователя

- [x] Панель вкладок рендерится горизонтально с одной кнопкой на опцию
- [x] Активная вкладка визуально выделена
- [x] Нажатие на вкладку вызывает колбэк `onChange` со значением вкладки
- [x] Активная вкладка плавно прокручивается в видимую область при переключении
- [x] Компонент поддерживает все три темы (Тёплая, Мягкая, Элегантная) через дизайн-токены

## Системные критерии приёмки

- [x] Компонент `Tabs` в `src/components/ui/Tabs/Tabs.tsx`
- [x] Обобщённый тип `<T extends string>` для типобезопасных значений вкладок
- [x] Пропсы: `options: TabOption<T>[]`, `value: T`, `onChange: (value: T) => void`, опциональный `className`
- [x] Интерфейс `TabOption<T>`: `{ value: T; label: string }`
- [x] Использует `role="tablist"` на контейнере, `role="tab"` и `aria-selected` на кнопках
- [x] CSS-модуль `Tabs.module.css` с классами `tabBar`, `tab`, `tabActive`, использующими дизайн-токены
- [x] Экспортируется через barrel `src/components/ui/index.ts` (компонент `Tabs` и тип `TabOption`)

## Связанные файлы

- `src/components/ui/Tabs/Tabs.tsx`
- `src/components/ui/Tabs/Tabs.module.css`
- `src/components/ui/index.ts`
