# 06-01: Система жестов карточек тренировки

Статус: Готово

## Описание

Переработка системы жестов смахивания на странице тренировочных карточек. Исправление бага с возвратом карточки, добавление 4-направленного смахивания (влево/вправо/вверх/вниз), реализация определения намерения на основе скорости с адаптивной физикой анимации, а также визуальные индикаторы зон при перетаскивании.

## Пользовательские критерии приёмки

- [x] Смахивание карточки влево, вправо, вверх или вниз отправляет её за экран в соответствующем направлении
- [x] Смахивание вправо записывает «Сказал!» (освоено)
- [x] Смахивание влево записывает «Пока нет» (не начато)
- [x] Смахивание вверх записывает «С подсказкой» (в процессе)
- [x] Смахивание вниз пропускает слово без записи результата
- [x] Быстрый бросок фиксирует смахивание даже при малом смещении
- [x] Медленное перетаскивание за порог фиксирует смахивание
- [x] Перетаскивание далеко в одну сторону и бросок обратно возвращает карточку в стопку (жест отмены)
- [x] Цветной оверлей с названием действия появляется на карточке при перетаскивании, усиливаясь к краю
- [x] Все три кнопки действий вызывают ту же анимацию вылета, что и жесты
- [x] Карточка отображается поверх заголовков и хлебных крошек при перетаскивании

## Системные критерии приёмки

- [x] Скользящее окно скорости (5 замеров, взвешенное 3:1 недавние к старым) заменяет одноточечное вычисление
- [x] Определение оси с горизонтальным смещением 1.2x предотвращает случайные вертикальные срабатывания
- [x] Приоритет скорости: при превышении порога направление определяется скоростью
- [x] Обнаружение отмены: скорость, противоположная значительному смещению (>40px), возвращает карточку
- [x] Адаптивная длительность анимации: `clamp(расстояние / скорость, 150мс, 400мс)`
- [x] Easing для импульса `cubic-bezier(0.2, 0, 0, 1)` для быстрых бросков
- [x] Easing для позиции `cubic-bezier(0.32, 0.72, 0, 1)` для медленных перетаскиваний
- [x] Сокращённое движение: длительность анимации 1мс
- [x] z-index стопки карточек на уровне `--z-sticky`
- [x] `SwipeCard` использует `forwardRef` с императивным хендлом `triggerSwipe`

## Критерии приёмки разработки

- [x] CSS Modules только с дизайн-токенами (цвета зон используют `--color-success`, `--color-danger`, `--color-warning`)
- [x] Прозрачность оверлея зон использует квадратичную кривую для плавного нарастания
- [x] Поддержка всех трёх тем (тёплая, мягкая, элегантная)
- [x] TypeScript strict, без ошибок линтера
- [x] Успешная продакшен-сборка
- [x] Все 160 существующих юнит-тестов проходят
