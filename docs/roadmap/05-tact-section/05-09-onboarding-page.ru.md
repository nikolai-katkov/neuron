# 05-09: Страница онбординга

Статус: Готово

## Описание

Добавить экран онбординга на уровне приложения, определяющий текущий уровень словарного запаса ребёнка перед доступом к основному приложению. Страница показывает три карточки уровней (начинающий, средний, продвинутый), определяющие начальный набор слов в словаре.

## Критерии приёмки для пользователя

- [x] При первом визите (нет состояния словаря в localStorage) вместо списка разделов отображается страница онбординга
- [x] Страница показывает заголовок "С чего начинаем?" и подзаголовок "Выберите, что лучше описывает вашего ребёнка сейчас"
- [x] Три карточки уровней появляются с поочерёдной анимацией входа (задержка 0.12с между карточками):
  - Начинающий (иконка Baby): "Только начинаем" / "Ребёнок пока не называет или называет 1--2 предмета"
  - Средний (иконка Sparkles): "Знает несколько слов" / "Ребёнок может назвать несколько знакомых предметов"
  - Продвинутый (иконка Star): "Называет многое" / "Ребёнок уверенно называет разные предметы в повседневной жизни"
- [x] Нажатие на карточку выделяет её (анимация масштабирования + свечения), затемняет остальные, завершает онбординг с задержкой 600мс
- [x] После выбора отображается основное приложение (список разделов); страница онбординга больше не появляется при последующих визитах
- [x] Весь текст доступен на обоих языках (EN и RU)

## Системные критерии приёмки

- [x] Компонент `OnboardingPage` в `src/pages/OnboardingPage.tsx`
- [x] Обёртка `OnboardingGate` в `App.tsx` проверяет `isOnboardingCompleted` из `useDictionary()`
- [x] Gate рендерит `OnboardingPage` при незавершённом онбординге, иначе рендерит `children`
- [x] Вызов `completeOnboarding(level)` инициализирует состояния слов на основе выбранного уровня:
  - beginner: только простые слова из мотивирующих категорий
  - intermediate: все простые слова по всем категориям
  - advanced: простые + средние слова по всем категориям
- [x] Выбранный уровень и флаг завершения сохраняются в localStorage через `DictionaryState`
- [x] Иконки из `lucide-react`: `Baby`, `Sparkles`, `Star`
