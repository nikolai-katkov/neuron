# Mom ABA

Приложение для помощи родителям в поддержке детей с аутизмом.

## Видение

Помочь родителям детей с аутизмом отслеживать этапы развития с помощью структурированных, научно обоснованных оценок, основанных на методологии вербальных оперантов ABA (прикладного анализа поведения).

См. [Статьи базы знаний](knowledge/README.ru.md) для документации по предметной области, начиная с [Оценки этапов развития](knowledge/developmental-milestones-assessment.ru.md).

## Технический обзор

- **Стек:** React 18, TypeScript (strict), Vite
- **Стилизация:** CSS Modules с системой дизайн-токенов
- **Тестирование:** Vitest + React Testing Library (юнит-тесты), Playwright (E2E)
