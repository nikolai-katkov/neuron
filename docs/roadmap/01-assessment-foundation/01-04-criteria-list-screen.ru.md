# 01-04: Экран списка критериев

Статус: Готово

## Описание

Экран прогрессии, показывающий 5 критериев раздела в виде карточек уровней. Каждая карточка отображает упрощённое название, краткое описание, статус и опциональный прогресс. Карточки упорядочены по уровню развития (1--5). Нажатие на карточку переходит к экрану оценки критерия.

## Критерии приёмки для пользователя

- [x] 5 карточек отображаются по порядку, каждая с: упрощённым названием, кратким описанием, значком статуса
- [x] Значки статуса: "Не начато" (по умолчанию), "В процессе", "Завершено"
- [x] Завершённые критерии визуально отличаются (галочка, изменение цвета или аналогичное)
- [x] Нажатие на любую карточку переходит к экрану оценки этого критерия
- [x] Общий прогресс раздела виден (например, "2/5 завершено" или индикатор прогресса)
- [x] Карточки отражают текущее состояние оценки (обновляются после возврата из оценки/тренировки)

## Системные критерии приёмки

- [x] Маршрут: `/sections/:sectionId/criteria`
- [x] Статус критерия читается из сохранённого состояния
- [x] Прогресс пересчитывается при монтировании и после возврата из навигации
- [x] Все 5 критериев доступны независимо от порядка выполнения (без принудительной линейности для MVP)

## Критерии приёмки для разработки

- [x] CSS Modules только с дизайн-токенами
- [x] Адаптивность для всех профилей устройств
- [x] Юнит-тест для расчёта прогресса (состояния 0/5, 3/5, 5/5)
- [x] Юнит-тест для отображения значков статуса по состоянию критерия
- [x] E2E-тест: навигация из введения, проверка отображения 5 карточек, нажатие переходит к оценке
